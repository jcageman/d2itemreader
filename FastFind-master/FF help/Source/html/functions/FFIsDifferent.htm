Function Reference

FFIsDifferent

Compares two SnapShots and determines whether they have any differences.

FFIsDifferent ( NoSnapShot1, NoSnapShot2 )

 

Parameters

NoSnapShot1 The "slot" number of the first SnapShot to compare.
NoSnapShot2 The "slot" number of the second SnapShot to compare.

 

Return Value

Success: Returns True
Failure: Returns False

 

Remarks

None.
